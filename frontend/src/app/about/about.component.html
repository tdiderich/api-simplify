API Bookie

API Monitoring for Humans
WHY

We all see an API and think, "Hey, I'd like to use that for (insert awesome idea here)." The only issue is that means we need to write the code for do the API monitoring, host a infrastructure to continurously run the code on, and maintain it. And, well, that just doesn't happen 90% of the time. That's where API Bookie comes in.

HOW
  • Input your API Endoint to Monitor
  • Setup your Authentication
  • Create the logic to use to montior the API Endpoint
USE CASES
  • Monitoring the BitCoin API for specific price points.
  • Monitoring your Enterprise SaaS Apps for outages.
  • Monitoring the a Weather API to be notified when it's going to rain.
